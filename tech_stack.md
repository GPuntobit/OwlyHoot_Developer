# Stack Tecnico - App OwlyHoot

## Linguaggio di Programmazione
- **Kotlin**: Linguaggio ufficiale per lo sviluppo Android, sicuro e moderno

## Framework UI
- **Jetpack Compose**: Toolkit moderno per creare interfacce utente native su Android

## Backend & Autenticazione
- **Firebase Authentication**: Gestione autenticazione utenti (email/password e OAuth)
- **Cloud Firestore**: Database NoSQL per archiviare dati in tempo reale
- **Firebase Storage**: Archiviazione file multimediali (immagini profilo, post)

## Database Locale
- **Room Database**: Libreria di astrazione SQLite per memorizzazione dati offline

## Wallet Integration
- **WalletConnect SDK**: Per connessione con portafogli esterni (Metamask, ecc.)
- **Web3J**: Libreria Java per interagire con blockchain Ethereum

## Altre Librerie Utilizzate
- **Retrofit/OkHttp**: Per chiamate API esterne
- **Gson/Moshi**: Serializzazione JSON
- **Coil**: Caricamento e caching immagini
- **DataStore**: Per memorizzare dati utente e preferenze
- **WorkManager**: Per eseguire operazioni in background
- **Navigation Component**: Per gestire la navigazione tra schermate
